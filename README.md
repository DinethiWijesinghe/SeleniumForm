This Python script demonstrates how to automate a sample web form using Selenium WebDriver, covering all major element-locating strategies including:
ID,
Name,
Tag Name,
CSS Selector,
Class Name,
XPath,
Relative Locators (to_right_of, below, near, above, to_left_of)

It interacts with text fields, password fields, textareas, dropdowns, checkbox, radio buttons, file upload, color picker, date picker, and finally submits the form.

Requirements

Python 3.8+  , Selenium 4+ , Chrome Browser

ChromeDriver (matching version) ,
Install Selenium:

pip install selenium

How to Run the Script

Save the Python file (e.g., selenium_form.py).

Ensure ChromeDriver is in PATH or same folder.

Run:

python selenium_form.py


The script will open a browser, automate the entire user interaction, and close the browser after submitting.



Purpose

This script is suitable for:

Selenium beginners learning element locating strategies

Demonstrating Selenium relative locators

Automation testing practice

Training and documentation
